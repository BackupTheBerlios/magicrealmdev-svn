Subversion

  • browse: http://svn.berlios.de/viewcvs/magicrealmdev/
  • anon access: svn checkout svn://svn.berlios.de/magicrealmdev/trunk
  • dev access:
    export SVN_SSH="ssh -l login-name"
    svn checkout svn+ssh://svn.berlios.de/svnroot/repos/magicrealmdev/trunk
    or, I either use cygwin's ssh, or putty/plink. To get the latter to work, I had to generate keys and register the pub key with svn.berlios.de before it would work.
    set SVN_SSH=plink -ssh -2 -l login-name
    there are no quotes in the set SVN_SSH command.
  • resources:
    • home page: http://subversion.tigris.org/
    • book: http://svnbook.red-bean.com/ CVS-users should read Appendix A. Subversion for CVS Users: http://svnbook.red-bean.com/en/1.0/apa.html
